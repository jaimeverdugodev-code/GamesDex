# 👾 GamesDex

**GamesDex** is a cross-platform social network designed for gamers to track their played games, write reviews, and discover new titles through an AI recommendation engine. 

Developed as a Final Degree Project (TFG) for Multiplatform Application Development (DAM).

## ✨ Key Features
* **Extensive Database:** Search and explore games utilizing the RAWG Video Games Database API.
* **Personal Backlog:** Keep track of the games you've played, are currently playing, or want to play.
* **Social Feed:** Follow other gamers, read their reviews, and see their latest activity.
* **Smart Recommendations:** Get personalized game suggestions powered by an AI model.
* **Cross-Platform:** Responsive web design built with Angular and Ionic, ready for mobile deployment via Capacitor.
* **Dark Mode UI:** Modern, gamer-focused dark theme with neon accents.

## 🛠️ Tech Stack
* **Frontend:** Angular, Ionic Framework, SCSS
* **Backend (BaaS):** Firebase (Authentication, Cloud Firestore)
* **External APIs:** RAWG API, Hugging Face Inference API (for recommendations)
